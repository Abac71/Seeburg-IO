1. Installing Raspberry PI OS
2. Enable SSH in raspberry configuration
3. XRDP install
	sudo apt-get install xrdp
	sudo systemctl status xrdp
4. git clone https://github.com/Abac71/Seeburg-IO.git
5. wiringPi install (for Raspberry Pi Zero 2 W)
	http://wiringpi.com/download-and-install/
	http://wiringpi.com/wiringpi-updated-to-2-52-for-the-raspberry-pi-4b/
	cd /tmp
	wget https://project-downloads.drogon.net/wiringpi-latest.deb
	sudo dpkg -i wiringpi-latest.deb
	gpio -v
6. Program code compile: gcc pi-seeburg.c -lwiringPi -o pi-seeburg
7. Samba install
	sudo apt-get update
	sudo apt-get install samba samba-common smbclient
	sudo mv /etc/samba/smb.conf /etc/samba/smb.conf_alt
	sudo nano /etc/samba/smb.conf
		[global]
		workgroup = WORKGROUP
		security = user
		encrypt passwords = yes
		client min protocol = SMB2
		client max protocol = SMB3
	testparm
	sudo service smbd restart
	sudo service nmbd restart
	sudo chown pi:pi /home/pi/Music
	sudo chmod 700 /home/pi/Music
	sudo nano /etc/samba/smb.conf
		[Samba3w1mp3]
		comment = Samba-3w1mp3-Freigabe
		path = /home/pi/Music
		read only = no
8. Blue-Alsa install https://www.raspberry-pi-geek.de/ausgaben/rpg/2018/04/musik-per-bluetooth-an-einen-lautsprecher-senden/
	sudo apt-get -y install bluealsa
	sudo systemctl disable bluealsa.service
	sudo cp /lib/systemd/system/bluealsa.service /etc/systemd/system
	testparm
	sudo service smbd restart
	sudo service nmbd restart
	sudo smbpasswd -a pi
8. Populate ./pi-seeburg using /etc/profile.
9. Configure automatic log-on with sudo raspi-config
